### Фичи Workflow

**Concurrency:**
```yaml
concurrency:
  group: ${{ github.workflow }}-${{ github.ref }}
  cancel-in-progress: true
```
→ Отменяет старые runs при новых push-ах (экономит compute)

**Caching:**
```yaml
cache: 'npm'
```
→ Кеширует npm dependencies (ускоряет установку)

**Matrix Strategy:**
```yaml
matrix:
  node-version: [20.x]
```
→ Можно легко добавить другие версии Node для тестирования

**Fail Conditions:**
- `continue-on-error: false` — блокирует pipeline
- `continue-on-error: true` — опциональный, не блокирует

### Triggers

Pipeline запускается на:
```yaml
on:
  push:
    branches: [main, develop]
  pull_request:
    branches: [main, develop]
```

✅ Push в main → CI запускается
✅ Push в develop → CI запускается
✅ PR к main → CI запускается
✅ PR к develop → CI запускается
✅ Несколько quick push-ей → только последний запущен (concurrency)
